# 문제: [카드 2][link]

[link]: https://www.acmicpc.net/problem/2164

### 출제자 : 정성헌

---
### 풀이법 : 

1주차 Queue 구현 문제 바탕으로 기존 클래스를 약간 수정하여 사옹하였습니다.

N장의 카드를 입력받아, 1번 카드가 제일 위에, N번 카드가 제일 아래인 상태로 순서대로 카드가 놓여있는데,

N=4인 경우를 예로 들면,

1 2 3 4 에서 

pop push pop 과정을 통해 

3 4 2 와 같이 변환됩니다.

따라서 카드가 1장이 남을때까지 반복문을 돌려

마지막 카드가 출력되게 합니다.
